# PawCare - Mobile Application

PawCare é um aplicativo desenvolvido para o gerenciamento completo da saúde e bem-estar do seu pet. O app permite acompanhar vacinas, consultas, histórico médico e muito mais.

## Funcionalidades
- **Dashboard**: Resumo das próximas atividades e status do pet.
*   **Perfil do Pet**: Informações detalhadas sobre o animal.
*   **Vacinas**: Controle de vacinação com histórico e próximas doses.
*   **Especialistas**: Busca de veterinários e clínicas parceiras.
*   **Histórico**: Registro completo de eventos de saúde.
*   **Cuidados**: Dicas e lembretes de cuidados diários.

## Tecnologias Utilizadas
- React Native
- Expo SDK 54
- Expo Router
- TypeScript

## Como executar
1. Clone o repositório
2. Instale as dependências: `npm install`
3. Inicie o projeto: `npm start`
